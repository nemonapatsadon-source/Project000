#clearScreen    ล้างหน้าจอ Console ให้สะอาด (รองรับทั้ง Windows และ Linux/macOS)
#waitEnter    หยุดรอให้ผู้เล่นกด Enter ก่อนจะดำเนินรายการต่อไป
#generateQuest    สุ่มสร้างภารกิจใหม่ (ชนิดพืช, อายุที่ต้องการ, และเงินรางวัล)
#stopGame    บันทึกสถานะเกมปัจจุบันแล้วแปลงเป็น Save Code และปิดโปรแกรม
#loadGame    รับรหัส Save Code มาถอดรหัสเพื่อคืนค่าสถานะผู้เล่น, แปลงเกษตร และไอเทม
#processNightEvents    คำนวณเหตุการณ์สุ่มตอนกลางคืน (ฝนตก, สัตว์บุก, ขโมยขึ้นบ้าน, หรือตรวจหวย)
#drawInterface    แสดงผล UI หลัก (สถานะผู้เล่น, รายละเอียดแปลงผัก, ภารกิจ และเมนูคำสั่ง)
#showIndex    แสดงคู่มือการเล่น ตารางราคาพืช และโอกาสสำเร็จในการเก็บเกี่ยว
#endDay    จบวัน: หักค่าเช่า, อัปเดตอายุพืช, รีเซ็ตค่า AP และเรียกใช้เหตุการณ์กลางคืน
#planting    จัดการการปลูกพืช: เลือกแถว, เลือกชนิดพืช และหักเงินค่าเมล็ดพันธุ์
#harvesting    จัดการการเก็บเกี่ยว: คำนวณโอกาสสำเร็จจาก ปุ๋ย/น้ำ/อายุ และเพิ่มเงินให้ผู้เล่น
#openShop    เมนูร้านค้า: ซื้อปุ๋ย, สปริงเกอร์, ขยายแปลง (ใช้ Gems), และอุปกรณ์ป้องกันฟาร์ม
#main    จุดเริ่มต้นของโปรแกรม: ใช้จัดการ Loop หลักของเกมและรับคำสั่งจากผู้เล่น
